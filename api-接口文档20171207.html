4.登录

  • BASE-URL + login

request:

参数 类型 说明

response:

{
    "userId": 123,
    "accountType": 1,// 账户类型
    "token": "fadfa13jkdjqo3ej13e1e12",

类型定义:

clientType 客户端类型 accountType 账号类型

2 安卓 2 管理员用户

3.录入新的运输数据

  • BASE-URL + insertTransportData

request:

参数 类型 说明
containerId String 柜号/车牌号
ladingId String 提单号(可空)
deviceId Long 设备号
transportType Int 运输类型
startPosition JSON 起运地{"country":"123","state":"124","city":"125","address":"XXXXX"}
endPosition JSON 目的地{"country":"126","state":"127","city":"128","address":"XXXXX"}
frequencyType String 频率类型(min/hour/day)
locateFrequency Int 定位频率(取值为数字)

response:

{
    "id": 123,// 后台唯一标识
}

类型定义:

frequencyType 定位频率类型 transportType 运输类型
min 分钟 0 火车
hour 小时 1 汽车
day 天 2 飞机
3 轮船

9.搜索

  • BASE-URL + search

request:(时间/起始地/目的地/运输类型/运输状态至少包含一种搜索条件)

参数 类型 说明
startDate Long 开始日期(yyyy-MM-dd 00:00:00)
endDate Long 截止日期(yyyy-MM-dd 23:59:59)
startPosition JSON 起始地{"country":"123","state":"124","city":"125"}
endPosition JSON 目的地{"country":"126","state":"127","city":"128"}
transportType int[] 运输类型[1,2,3]
transportStatus int 运输状态(不传表示全部返回全部状态)
page int 页码
pageCount int 每页数据个数

response: 同 运输列表
